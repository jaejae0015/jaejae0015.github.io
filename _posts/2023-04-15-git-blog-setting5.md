1. 개인의 <a href="https://search.google.com/search-console">google Search console</a>

## 구글 검색엔진 등록하기!

아무리 검색을 해도, 나의 GIT Blog는 나오지 않는다.

아래와 같이 따라해서 나오도록 해보자:)

1.<a href="https://search.google.com/search-console">google Search console</a>에 접속 후, URL 검사에 들어간다.

![image](https://user-images.githubusercontent.com/56392513/231962953-995e04f3-e94d-411c-acf2-4c4a089968a5.png)

2.본인이 등록하고자 하는 URL 입력 후, 파일 색인 요청을 진행한다.

![image](https://user-images.githubusercontent.com/56392513/231966549-9cb533f3-97c3-432d-a41c-5437d00c984d.png)

3.색인 요청 완료 후, 바로 반영이 안된다. 24시간 후에 확인 하면 아래와 같이 검색이 된다.

![image](https://user-images.githubusercontent.com/56392513/231967028-a19aad65-99f0-4345-b719-2f6030f4fe26.png)
